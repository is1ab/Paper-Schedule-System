# Paper Schedule System

## Installation

### Python Environment

You need a Python environments first.

```
python3 -m pip install -r requirements.txt
```

### Firebase Credential

You need a Firebase Credential and place it under `api/` folder.

### Run Application

#### Backend

Run the commands. 

```
cd api
flask run -h 0.0.0.0 -p 8080
```

#### Frontend

Run the command

```
cd app
npm install
npm run dev
```
